# 🌐 CodeAlpha - Web Scraping Project

## 📌 Project Description

This project was developed as part of my **Data Analytics Internship at CodeAlpha**. The objective of this project is to extract data from a website using Python libraries and convert the extracted information into a structured dataset for further analysis.

The project demonstrates the practical implementation of:

* Web Scraping
* HTML Parsing
* Data Collection and Storage
* Dataset Creation using Python

---

## 🎯 Objective

To scrape quotes and author information from a public website and generate a custom dataset in CSV format.

---

## 🛠️ Technologies Used

* Python 3
* Requests
* BeautifulSoup4
* Pandas
* Visual Studio Code
* Git & GitHub

## 📊 Dataset Information

The generated dataset contains the following fields:

| Column Name | Description        |
| ----------- | ------------------ |
| Quote       | Text of the quote  |
| Author      | Name of the author |
